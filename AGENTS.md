This is a monorepo for Bonjourr Startpage API. It contains `weather` for weather & `services` for everything else.

# Submodules

This repo contains the following git submodules. Never modify files inside them — changes must go to their respective upstream repositories.

| Path | Repository |
|------|------------|
| `weather/meteo` | https://github.com/victrme/racle-meteo |
| `services/src/suggestions` | https://github.com/victrme/search-suggestions |
| `services/src/favicon` | https://github.com/victrme/favicon-fetcher |
| `services/src/quotes` | https://github.com/victrme/i18n-quotes |
