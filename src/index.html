Bonjourr APIs

Unsplash

This is a proxy of the unsplash endpoints. Read their docs to learn more: unsplash.com/documentation. When cloning this API, add your key with "UNSPLASH" as an environnement variable.

Endpoint

GET /unsplash/[unsplash-endpoints]

Example

  • /unsplash/photos/random

Weather

This is a mostly a proxy of openweathermap that combines their current & forecast endpoints. Use their docs here: openweathermap.org/current. Go to GET /weather/current for current weather, and GET /weather/forecast for the forecast.

Endpoints

GET /weather/current?[openweathermap-parameters]
GET /weather/forecast[openweathermap-parameters]

Examples

  • /weather/current?q=Paris,FR
  • /weather/forecast?q=Paris,FR

Favicon

Fetches the URL of a website's favicon. Add the URL of the website you want (with the protocol!) to the API path.
Source code: favicon-fetcher

Endpoint

GET /favicon/:your-website

Example

  • /favicon/https://victr.me

Quotes

Serves quotes in different languages, but also inspirobot and kaamelott quotes. For classic quotes, if the language is not available, it defaults to english.
Source code: i18n-quotes

Endpoints

GET /quotes/classic/:lang

GET /quotes/inspirobot

GET /quotes/kaamelott

Examples

  • /quotes/classic/fr
  • /quotes/inspirobot
  • /quotes/kaamelott

Suggestions

This is a scraping tool that returns search suggestions from different providers. It uses websockets for cost reasons. Use ws://. Docs is TODO.
Source code: search-suggestions

Endpoint

WS /suggestions

Example

/suggestions?with=google&q=minecraft
