Bonjourr APIs

Unsplash

This is a proxy of the unsplash endpoints. Read their docs to learn more: unsplash.com/documentation

Example

/unsplash

Weather

This is a mostly a proxy of openweathermap that combines their current & forecast endpoints. Use their docs here: openweathermap.org/current. Go to GET /weather/current for current weather, and GET /weather/forecast for the forecast.

Example

/weather/current?q=Paris,FR

/weather/forecast?q=Paris,FR

Favicon

Fetches the URL of a website's favicon. Add the URL of the website you want (with the protocol!) to the API path.

Example

/favicon/https://victr.me

Quotes

Serves quotes in different languages, but also inspirobot and kaamelott quotes. Go to GET /quotes/classic/[lang] to get classic quotes. If the language is not available, it defaults to english. GET /quotes/inspirobot for inspirobot, GET /quotes/kaamelott for kaamelott quotes

Example

/quotes/classic/fr

/quotes/inspirobot

/quotes/kaamelott

Suggestions

This is a scraping tool that returns search suggestions from different providers. It uses websockets for cost reasons. Docs is TODO

Example

/suggestions?with=google&q=minecraft
